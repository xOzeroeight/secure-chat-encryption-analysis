# secure-chat-encryption-analysis
Performance analysis of AES, RSA, and ChaCha20 encryption algorithms for secure chat applications using Python.

## Dataset

The text samples used in this project were derived from the Amazon Fine Food Reviews dataset on Kaggle:

https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews

The original dataset was used to extract text samples, which were grouped into small, medium, and large input files for encryption benchmarking.
